# سكّر متزن - Sukar Balanced

تطبيق شامل لإدارة سكر الدم والإنسولين والوجبات، مصمم خصيصاً للمستخدمين الناطقين بالعربية.

## 🎯 الميزات الرئيسية

### 1. إدارة قراءات السكر
- تسجيل قراءات سكر الدم يومياً
- تصنيف القراءات (آمنة، مرتفعة، منخفضة)
- عرض الإحصائيات اليومية والمتوسطات
- إضافة ملاحظات مع كل قراءة

### 2. حاسبة الإنسولين الذكية
- حساب جرعة الإنسولين بناءً على:
  - مستوى السكر الحالي
  - كمية الكربوهيدرات المتناولة
- معادلة طبية معتمدة
- حفظ السجل التاريخي للحسابات

### 3. إدارة الوجبات
- قاعدة بيانات عربية تحتوي على 30+ صنف طعام
- معلومات غذائية كاملة (كربوهيدرات، بروتين، دهون، سعرات)
- إمكانية إضافة وجبات مخصصة
- تتبع الوجبات اليومية

### 4. السجل التاريخي
- عرض جميع القراءات والوجبات
- تصفية حسب التاريخ
- حذف السجلات القديمة
- تصدير البيانات

### 5. الإشعارات المحلية
- تذكيرات بقياس السكر كل 4 ساعات
- تذكيرات بأوقات الدواء (صباح ومساء)
- الإشعارات تعمل حتى لو كان التطبيق مغلقاً

### 6. التخزين المحلي
- جميع البيانات تُحفظ محلياً على الجهاز
- عدم الحاجة لاتصال إنترنت مستمر
- الخصوصية والأمان الكامل

## 🚀 البدء السريع

### 1. تثبيت التبعيات
```bash
pnpm install
```

### 2. تشغيل التطبيق في وضع التطوير
```bash
pnpm dev
```

### 3. بناء التطبيق للإنتاج
```bash
pnpm build
```

### 4. بناء APK (اطلع على BUILD_INSTRUCTIONS.md)
```bash
cd android
./gradlew assembleDebug
```

## 🔧 التقنيات المستخدمة

### Frontend:
- **React 19**: مكتبة واجهات المستخدم
- **TypeScript**: لغة البرمجة المكتوبة بقوة
- **Tailwind CSS 4**: نظام التصميم
- **Wouter**: توجيه العميل
- **shadcn/ui**: مكونات واجهة المستخدم

### Mobile:
- **Capacitor**: إطار عمل لتحويل تطبيقات الويب إلى تطبيقات أصلية
- **Capacitor Local Notifications**: الإشعارات المحلية
- **Capacitor Preferences**: التخزين المحلي

### Build:
- **Vite**: أداة بناء سريعة
- **Gradle**: نظام بناء أندرويد
- **Node.js**: بيئة التطوير

## 📝 ملاحظات مهمة

### الخصوصية:
- جميع البيانات تُحفظ محلياً على الجهاز
- لا يتم إرسال أي بيانات إلى خوادم خارجية
- التطبيق يعمل بشكل كامل بدون إنترنت

### الأمان:
- كلمات المرور محمية محلياً
- البيانات الحساسة لا تُنقل عبر الشبكة
- لا توجد تسجيلات بيانات شخصية

### الأداء:
- التطبيق محسّن للعمل على أجهزة بمواصفات منخفضة
- حجم التطبيق صغير نسبياً
- استجابة سريعة للعمليات
